About

I built this page as a way to browse addons from the web. The generation script is basically some python glue encompassing many different helpful tools.

  • Jinja2 for html templating.
  • HTML5 Boilerplate for the build scripts which concat and minify resources and cut down on .png image size.
  • Isotope for the slick animations (and the colorscheme as well!)
  • LessFramework for column width inspiration.
